# Project Installation Guide

- Download Volta [link](https://volta.sh/)
- Download Postgres `16.2`

<br>

# Tech Stack

- Frontend : `React`
- Backend: `Node`
- Database: `Postgres`

<br>

# Installation Instructions

1. Download the code from Repository
1. Run npm i in root and frontend folder
1. Check the `package.json` file for commands
